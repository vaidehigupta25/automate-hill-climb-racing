# Automate Hill Climb Racing

This project automates the gameplay of **Hill Climb Racing**, a popular physics-based driving game, using Python. 
The automation leverages **computer vision** and **keyboard control** to perform actions in the game automatically.

## Features
- Automatic car control based on game state.
- Detects obstacles and manages jumps.
- Simple and lightweight Python-based solution.
- Works with BlueStacks or LDPlayer Android emulators.

## Technologies Used
- Python 3.x
- OpenCV (for image processing)
- pyautogui (for keyboard/mouse automation)
- Mediapipe (optional for gesture control)

## How to Use
1. Install [BlueStacks](https://www.bluestacks.com/) or [LDPlayer](https://www.ldplayer.net/).
2. Open Hill Climb Racing on the emulator.
3. Run the Python script:
   ```bash
   python automate_hill_climb.py
